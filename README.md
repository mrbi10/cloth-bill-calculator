## What this app does

* Calculates required cloth length based on item quantity
* Applies stitching rates per item
* Gives a clear final bill output
* Designed to be easy to extend for printing and shop branding

## Getting started (local setup)

Clone the repo:

```bash
git clone https://github.com/mrbi10/cloth-bill-calculator.git
cd cloth-bill-calculator
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm start
```

Open in browser:
[http://localhost:3000](http://localhost:3000)

---

## Available scripts

### `npm start`

Runs the app in development mode.
The page reloads on changes.

### `npm run build`

Creates an optimized production build in the `build` folder.

### `npm test`

Runs tests in watch mode (if added later).

### `npm run deploy`

Builds the app and deploys it to GitHub Pages.

---

## Deployment (GitHub Pages)

This project is already configured for GitHub Pages.

If you fork or clone and want to deploy again:

```bash
npm run deploy
```

Make sure GitHub Pages is set to:

* Branch: `gh-pages`
* Folder: `/root`
